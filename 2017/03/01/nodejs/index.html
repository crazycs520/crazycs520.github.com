Node.JS

##Node.JS 简介

  • 共享状态的并发。注意回调函数和错误处理修改内存中的变量（状态）。
  • 阻塞与非阻塞。nodejs采用事件轮询，故setTimeout()是非阻塞的，异步的。
  • Node并发采用事件轮询。
  • 单线程。调用（执行）堆栈
  • 错误处理。
  • 堆栈追踪

##Node 中的 API

  • 全局对象 global
  • 模块系统的三个核心对象:require,module,export
  • 事件，监听。
  • buffer
  • stream
  • 监视
  • 事件驱动 events
    • events.EventEmitter
